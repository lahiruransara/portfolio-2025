Overview
CORONA Admin Unofficial Design System
Corona Admin template flaunts a simple and unique design along with an intriguing dark and shadowy color scheme. It contains a powerful dashboard and is packed with features and useful components capable of simplifying complex web app development projects. Data representation and visualization are no longer challenging, as the template offers multiple options for the same.
This Design system was built using atomic design principles and has captured most of the components.
The Goal
Create a scalable design system of components with proper documentation that can create a cohesive design experience for users as well as streamline development work.
Future Work
Looking towards the future
As the design system grows, I hope to create more standardized assets that can be reused in many ways. I hope to break down the the components and document them with how the component behaves.
